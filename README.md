# support-tool
Lightweight help desk application built with ruby on Rails for tracking and managing internal support requests.

## Tech Stack
- Ruby on Rails
- PostgreSQL
- ActiveRecord

## Features (WIP)
- User creation and management
- Ticket creation and tracking
- Basic CRUD functionality

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/support-tool.git
cd support-tool

bundle install
bin/rails db:create
bin/rails db:migrate
bin/rails server
